Pseudoclases

Se identifican con :

Las Pseudoclases sirven para dar una accion especifica a un elemento como cambiar su color o para seleccionar un elemento en especifico.

Pseudoelementos

Se identifican con ::

Los Pseudoelementos sirven para seleccionar elementos especificos como una linea o la primera letra como tambien para agregar un elemento al inicio ::before o despues ::after
